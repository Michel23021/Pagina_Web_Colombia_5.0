# 🇨🇴 Colombia 5.0 — Sitio Web Informativo Bilingüe

Sitio web informativo desarrollado como actividad académica para el **SENA Centro de Manufactura en Textil y Cuero**, documentando el evento tecnológico **Colombia 5.0** (Bogotá, 8 de mayo 2026).

## 🌐 Ver el sitio

## 📋 Contenido del sitio

| Sección | Descripción |
|---|---|
| **Inicio** | Presentación del evento con métricas clave |
| **Conferencias** | 2 conferencias con resúmenes en español e inglés |
| **Galería** | 18 fotos del evento con lightbox interactivo |
| **Glosario** | 30+ términos técnicos ES/EN con búsqueda en tiempo real |
| **Ética** | Reflexión sobre ética en tecnología empresarial |

## 🛠 Tecnologías utilizadas

- HTML5
- CSS3 (Custom Properties, Grid, Flexbox, Animations)
- JavaScript Vanilla (IntersectionObserver, DOM manipulation)
- Google Fonts (Bebas Neue, Syne, DM Sans)
