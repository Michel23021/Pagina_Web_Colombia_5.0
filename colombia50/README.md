# Colombia 5.0 — Sitio Web Informativo Bilingüe

Proyecto académico desarrollado para el SENA – Centro de Manufactura en Textil y Cuero, Regional Distrito Capital.

## 📋 Descripción

Sitio web informativo bilingüe (español / inglés) sobre el evento tecnológico **Colombia 5.0**, desarrollado con tecnologías FrontEnd puras.

## 🛠️ Tecnologías utilizadas

- HTML5
- CSS3 (Diseño responsive, Variables CSS, Animaciones)
- JavaScript (Vanilla JS)

## 📁 Estructura del proyecto

```
colombia50/
├── index.html              ← Página de inicio
├── css/
│   └── style.css           ← Estilos globales
├── js/
│   └── main.js             ← Interactividad JS
├── img/                    ← Imágenes del evento
└── pages/
    ├── conferencias.html   ← Página de conferencias (bilingüe)
    ├── glosario.html       ← Glosario técnico en inglés (35 términos)
    └── etica.html          ← Reflexión ética y conclusión
```

## 📄 Secciones del sitio

| Página | Contenido |
|--------|-----------|
| `index.html` | Nombre del evento, introducción, objetivo del sitio |
| `conferencias.html` | 2 conferencias con resumen bilingüe, keywords y multimedia |
| `glosario.html` | 35 términos técnicos EN → ES con definición + buscador |
| `etica.html` | Reflexión ética: IA, datos, automatización, impacto social |
