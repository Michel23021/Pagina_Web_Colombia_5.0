# Colombia 5.0 – Memorias del Evento Tecnológico

¡Bienvenido al repositorio oficial de **Colombia 5.0**! Este es un sitio web bilingüe e interactivo desarrollado como una actividad académica para el **SENA Centro de Manufactura en Textil y Cuero (CMTC)** en Bogotá. El proyecto documenta las conferencias, experiencias y aprendizajes adquiridos durante el evento tecnológico, aplicando un diseño futurista de alto impacto visual.

---

## 🌌 Estética y Concepto Visual

El sitio web está implementado bajo una dirección de arte **Cyberpunk + Glassmorphism**, inspirada en las interfaces de usuario de última generación. 

### Paleta de Colores
*   **Fondo:** Oscuro profundo (`#0d0e15`) para maximizar el contraste del neón.
*   **Acentos:** Morado eléctrico, azul neón y detalles en oro/dorado metálico.
*   **Efectos:** Tarjetas con efecto de "cristal esmerilado" mediante desenfoque de fondo (`backdrop-filter: blur`) y bordes sutiles iluminados.

---

## 🛠️ Características Principales

*   **🌐 Arquitectura Bilingüe Completa (ES / EN):** Conmutación de idioma instantánea y nativa (sin recargar la página) mediante atributos `data-lang` y JavaScript, abarcando la navegación, descripciones, resúmenes y términos técnicos.
*   **📑 Navegación Modular Distribuid:**
    *   `index.html`: Página principal con la introducción, el bloque cronológico de conferencias y la galería multimedia.
    *   `ethics.html`: Sección independiente dedicada a la reflexión ética sobre la Inteligencia Artificial, el uso de datos y el impacto social en Colombia.
    *   `glossary.html`: Glosario técnico interactivo.
*   **🔍 Buscador en Tiempo Real:** Filtro predictivo integrado en el glosario que escanea filas tanto en inglés como en español simultáneamente mientras el usuario escribe.
*   **📸 Galería Optimizada:** Estructura responsiva preparada para renderizar el registro fotográfico cronológico del evento.
