Technical Glossary

Más de 30 términos técnicos en inglés aprendidos durante el evento Colombia 5.0, con traducción al español y definición.

English Español Definition / Definición
Artificial Intelligence (AI) Inteligencia Artificial (IA) Simulation of human intelligence by machines, especially computer systems, to perform tasks that normally require human cognition.
Machine Learning Aprendizaje Automático Branch of AI that enables systems to learn and improve from experience without being explicitly programmed.
Deep Learning Aprendizaje Profundo Subset of machine learning using neural networks with many layers to analyze complex patterns in data.
Cloud Computing Computación en la Nube Delivery of computing services (servers, storage, databases, networking) over the internet on demand.
Big Data Grandes Datos Extremely large datasets that require advanced tools to capture, store, analyze and visualize effectively.
API (Application Programming Interface) Interfaz de Programación de Aplicaciones Set of definitions and protocols that allows different software applications to communicate with each other.
DevOps Desarrollo y Operaciones Methodology combining software development and IT operations to shorten development cycles and improve delivery quality.
Cybersecurity Ciberseguridad Practice of protecting systems, networks and programs from digital attacks, damage, or unauthorized access.
Blockchain Cadena de Bloques Distributed ledger technology that records transactions in a secure, transparent, and immutable chain of blocks.
Internet of Things (IoT) Internet de las Cosas Network of physical devices embedded with sensors and software that collect and exchange data over the internet.
Automation Automatización Technology that performs tasks with minimal human intervention, increasing efficiency and reducing errors.
Data Science Ciencia de Datos Interdisciplinary field that uses statistics, algorithms, and machine learning to extract insights from data.
Agile Ágil Iterative approach to software development that promotes flexibility, collaboration, and rapid delivery of functional software.
Microservices Microservicios Architectural style that structures an application as a collection of small, independently deployable services.
Neural Network Red Neuronal Computing system modeled after the human brain, consisting of interconnected nodes that process information in layers.
Prompt Engineering Ingeniería de Prompts Practice of designing and optimizing input instructions to get desired outputs from AI language models.
Digital Transformation Transformación Digital Integration of digital technology into all areas of a business, fundamentally changing how it operates and delivers value.
Scalability Escalabilidad Capability of a system or process to handle a growing workload by adding resources without losing performance.
Encryption Cifrado Process of encoding information so only authorized parties can access it, protecting data from unauthorized access.
Phishing Suplantación de Identidad Fraudulent attempt to steal sensitive information by disguising as a trustworthy entity in electronic communications.
Ransomware Secuestro de Datos Type of malicious software that encrypts a victim's files and demands payment for the decryption key.
Multi-Factor Authentication (MFA) Autenticación Multifactor Security method requiring users to verify identity through two or more independent credentials before access is granted.
Zero Trust Confianza Cero Security model that requires strict verification of every user and device, regardless of location, before granting access.
Generative AI IA Generativa AI systems capable of creating new content (text, images, audio, code) by learning patterns from existing data.
Large Language Model (LLM) Modelo de Lenguaje Grande AI model trained on massive amounts of text data to understand and generate human-like language.
Fintech Tecnología Financiera Technology that improves and automates financial services, making them more accessible and efficient for businesses and consumers.
Startup Ecosystem Ecosistema Emprendedor Community of startups, investors, mentors, and institutions that interact to support entrepreneurship and innovation.
Open Source Código Abierto Software whose source code is freely available for anyone to use, modify, and distribute.
Data Privacy Privacidad de Datos Right of individuals to control how their personal information is collected, stored, and used by organizations.
Algorithmic Bias Sesgo Algorítmico Systematic errors in AI outputs that create unfair outcomes due to flawed assumptions in the training data or algorithm design.
Software as a Service (SaaS) Software como Servicio Cloud-based software delivery model where applications are hosted remotely and accessed via the internet on subscription.
User Experience (UX) Experiencia de Usuario Overall experience a person has when interacting with a product, particularly in terms of usability and satisfaction.
Responsive Design Diseño Responsivo Web design approach that makes pages render well on all devices and screen sizes, from desktop to mobile.
Version Control Control de Versiones System that tracks and manages changes to source code over time, allowing teams to collaborate effectively (e.g., Git).
